首次使用前，请先完成以下准备：

  1. 前往 Google Cloud Console 创建项目
  2. 启用 AdSense Management API
  3. 配置 OAuth 同意屏幕，添加 scope: adsense.readonly
  4. 创建 OAuth 凭证 (Web Application)，重定向 URI 填: http://localhost:8000/oauth/callback
  5. 下载 JSON 并重命名为 client_secret.json 放入 credentials/ 目录
  6. 点击上方按钮完成授权
